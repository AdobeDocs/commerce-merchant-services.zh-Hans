# 推荐类型

Adobe Commerce提供了大量可部署到网站上各个页面的推荐。 所有推荐类型都由数据驱动。 它们由行为数据、产品属性数据和量度提供支持。 为便于参考，推荐类型按以下方式分组：

- [个性化](#personalized)
- [交叉销售和向上销售](#crossup)
- [热门程度](#popularity)
- [高性能](#highperf)

作为最佳实践，Adobe在使用推荐时建议遵循以下准则：

- 使推荐类型多样化。 如果客户一次又一次地提出相同的产品建议，则会开始忽略推荐。

- 请勿将相同的推荐部署到购物车页面和订单确认页面。 请考虑使用 `Most Added to Cart` 购物车页面和 `Bought This, Bought That` ，用于订单确认页面。

- 保持网站整洁。 请勿在同一页面上部署三个以上的推荐单元。

- 如果你的店卖衣服， `More like this` 建议可建议与所查看产品的性别不匹配的特定于性别的产品。 请考虑仅对非服装类别使用此推荐类型。

## 个性化 {#personalized}

这些推荐类型会根据您网站上特定购物者的行为历史记录推荐产品。

| 类型 | 描述 |
|---|---|
| 建议您 | 根据每位购物者的当前和以前的现场行为推荐产品。 根据购物者的浏览和购买历史记录显示高度相关的推荐。 此推荐类型对于大多数购物者开始在网站上的旅程的主页非常有效。 对于您网站上首次没有生成任何信号以个性化其体验的购物者，Adobe Commerce会根据“查看次数最多”的推荐类型显示产品。 但是，当购物者开始与网站上的产品进行交互时，推荐的产品会根据其行为实时进行调整。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别&#x200B;<br/><br/>**建议的标签：**<br/>  — 只为您<br/> — 建议您<br/> — 受您的购物趋势的启发 |
| 最近查看的 | 根据浏览器历史记录显示购物者最近查看的产品。 推荐单元会删除所有已删除的产品。 如果没有浏览器历史记录，或者在应用过滤器规则时没有足够的历史记录，则不会显示推荐单元。 如果结果包含的产品少于配置的产品，则推荐单元仅显示返回的产品。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/> — 最近查看<br/> — 再看看 |

## 交叉销售和向上销售 {#crossup}

这些推荐类型是社交推荐类型，旨在帮助购物者找到他人喜欢的产品或产品驱动的产品，以帮助他们找到其他类似的产品

| 类型 | 描述 |
|---|---|
| 已查看此内容，已查看 | 推荐购物者使用当前查看的产品查看频率过高的产品。<br/><br/>**使用位置：**<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/> — 查看了此产品的客户也查看了(PDP) |
| 已查看此内容，但已购买 | 推荐在查看当前产品后，购物者往往会更频繁地购买的产品。 可帮助引导购物者发现他们可能没有注意到的产品。<br/><br/>**使用位置：**<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/> — 查看了此最终购买产品的客户<br/> — 最终购买的客户<br/> — 查看此产品后，其他人会购买什么？ |
| 买了这个，买了那个 | 建议购买者使用当前查看的产品更频繁地购买的产品。 在购物车或产品详细信息页面上使用的最常用方法，是增加相关交叉销售产品的曝光率，以提高平均订单价值。 显示高度相关的产品，购物者可以通过聚合其他购物者使用当前产品购买的内容将其添加到购物车。<br/><br/>**使用位置：**<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/> — 获取您所需的一切<br/> — 不要忘记这些<br/> — 经常一起购买 |
| 更像这样 | 推荐基于相似元数据（如名称、描述、类别分配和属性）的产品。 通过评估所查看产品的属性，可在同一类别中推荐类似产品。 例如，如果购物者正在浏览瑜伽垫，则建议使用设备类别中的其他产品。 由于此推荐类型不区分性别，因此不建议在服装、时尚或其他特定于性别的垂直行业中使用。<br/><br/>**使用位置：**<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/>  — 更多此类产品<br/> — 类似于 |
| [视觉相似度](#visualsim) | 建议将外观相似的产品与正在查看的产品进行比较。 如果图像和产品的视觉方面对购物体验至关重要，则此推荐类型非常有用。 |

## 热门程度 {#popularity}

这些推荐类型建议的产品是过去七天内最受欢迎或最趋势的产品。

| 类型 | 描述 |
|---|---|
| 查看次数最多 | 建议通过计算过去七天内发生查看操作的会话数来查看次数最多的产品。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/> — 最受欢迎<br/> — 趋势<br/> — 当前受欢迎<br/> — 最近受欢迎<br/> — 受此产品(PDP)启发的流行产品<br/> — 最畅销商品 |
| 购买次数最多 | 建议购物者在过去七天内最常购买的产品。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/>  — 最受欢迎<br/> — 趋势<br/> — 当前受欢迎<br/> — 最近受欢迎<br/> — 受此产品(PDP)启发的流行产品<br/> — 最畅销商品 |
| 添加到购物车的次数最多 | 建议购物者在过去七天内最常添加到购物车的产品。 此推荐类型可在所有页面上使用。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/>  — 最受欢迎<br/> — 趋势<br/> — 当前受欢迎<br/> — 最近受欢迎<br/> — 受此产品(PDP)启发的流行产品<br/> — 最畅销商品 |
| 趋势 | 根据产品在您网站中的受欢迎程度最近的趋势推荐产品。<br/><br/>Adobe Sensei可汇总您网站内的浏览和购买数据，以确定和排名哪些产品最近最受您的购物者欢迎。 由于趋势分析最近的产品势头，因此对于周转率较高的目录而言，它是一种有效的推荐类型。 如果您的目录比较静态，则它可能没有那么有用，除非受众的购物模式是高度变量。<br/><br/>当在主页上使用时，趋势分析会推荐最近在整个网站中很受欢迎的产品。 趋势不显示一直受欢迎的产品，而是显示最近受欢迎的产品。 例如，如果您有一个电子邮件营销活动，用于推广某些产品，则电子邮件产生的热门程度增加会增加促销产品分类为趋势的可能性。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/> — 趋势<br/> — 立即显示趋势<br/> — 最近趋势<br/> — 热门产品<br/> — 趋势相关产品(PDP) |

## 高性能 {#highperf}

这些推荐类型根据成功标准（如添加到购物车或转化率）推荐性能最佳的产品。

| 类型 | 描述 |
|---|---|
| 查看购买转化 | 推荐具有最高查看到购买转化率的产品。 在注册产品查看的所有购物者会话中，最终由购物者注册购买的比例是多少。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/>  — 最畅销商品<br/> — 热门产品<br/> — 您可能对 |
| 查看购物车转化 | 推荐具有最高查看到购物车转化率的产品。 在注册产品查看的所有购物者会话中，购物者最终注册购物车加货的比例是多少。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/>  — 最畅销商品<br/> — 热门产品<br/> — 您可能对 |
| 购买次数最多 | 此推荐类型通常称为“最畅销商品”，用于计算在过去七天内发生下单操作的会话数。 此推荐类型可在所有页面上使用。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/>  — 最受欢迎<br/> — 趋势<br/> — 当前受欢迎<br/> — 最近受欢迎<br/> — 受此产品(PDP)启发的流行产品<br/> — 最畅销商品 |
| 添加到购物车的次数最多 | 建议购物者在过去七天内最常添加到购物车的产品。 此推荐类型可在所有页面上使用。<br/><br/>**使用位置：**<br/> — 主页<br/> — 类别<br/> — 产品详细信息<br/> — 购物车<br/> — 确认&#x200B;<br/><br/>**建议的标签：**<br/>  — 最受欢迎<br/> — 趋势<br/> — 当前受欢迎<br/> — 最近受欢迎<br/> — 受此产品(PDP)启发的流行产品<br/> — 最畅销商品 |

## 视觉相似度 {#visualsim}

的 _视觉相似度_ 推荐类型建议将外观相似的产品与正在查看的产品进行比较。 当产品的图像和视觉方面是购物体验的重要部分时，此推荐类型非常有用。

### 工作原理

的 _视觉相似度_ 推荐类型为目录中与当前查看的图像具有视觉相似度的其他产品提供了推荐。 视觉相似性包括以下方面：

- 颜色
- 形状
- 大小
- 纹理
- 材料
- 样式

Adobe Sensei使用AI处理和分析目录中的图像，并构建用于确定视觉相似性的属性。

>[!NOTE]
>
> 如果您在非生产环境中测试此推荐类型，请确保您的图像URL是可公开访问的。

>[!NOTE]
>
> 目前，产品映像的大小必须为10 MB或更小。

由于此推荐类型不适用于大多数目录，因此默认情况下未启用此类型。 您必须明确启用此推荐类型。

### 启用可视相似度推荐类型

>[!NOTE]
>
> 的 _视觉相似度_ 推荐类型在您 [安装](install-configure.md) 作为可选模块。

1. 在 _管理员_ 侧栏，转到 **营销** > _促销活动_ > **产品Recommendations** 以显示 _产品Recommendations_ 功能板。

1. 单击 **设置** （齿轮图标） _设置_ 页面。

1. 在 _可视化Recommendations_ 选择 **启用可视化Recommendations**.

1. 单击 **保存更改** 完成。

   的 [创建新推荐](create.md) 页面立即显示 **视觉相似度** 页面类型为 **产品详细信息**.

启用可视化推荐后，Adobe Sensei会启动图像处理。 时间长短取决于目录的大小。
